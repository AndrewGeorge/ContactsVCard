ContactsVCard
=============

基于android的联系人电子名片生成工具----开始于2014-01-11

#如何使用
打开`adt-bundle-android`的eclipse，然后导入该工程，注意在`项目属性中导入libs文件夹下的库文件`

项目中用了android 3.0之后的特性，请不要在3.0之前的版本上使用
